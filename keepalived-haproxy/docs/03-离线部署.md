# 离线部署

Keepalived 和 HAProxy 是 OS 包，不包含在 K3s 官方离线包里。

离线环境需要提前准备目标系统匹配的包：

- `haproxy`
- `keepalived`
- `iproute` / `iproute2`
- `iptables`
- `procps-ng` 或提供 `killall` 的包

安装好 OS 包后执行：

```bash
LB_AIRGAP=true \
bash keepalived-haproxy/scripts/deploy-keepalived-haproxy.sh keepalived-haproxy/prod.env
```

`LB_AIRGAP=true` 时脚本不会尝试访问 yum/apt/dnf，只检查命令是否存在。
