# Keepalived + HAProxy

本目录描述基于 Keepalived 和 HAProxy 的高可用 TCP 入口。

在当前 K3s 3 master 方案中，它提供 Kubernetes API 的稳定入口：

```text
client -> VIP:8443 -> HAProxy -> k3s master:6443
```

## 文件

```text
keepalived-haproxy/
├── 说明.md
├── config.example.env
├── docs/
│   ├── 01-架构说明.md
│   ├── 02-在线部署.md
│   ├── 03-离线部署.md
│   └── 04-运维手册.md
└── scripts/
    └── deploy-keepalived-haproxy.sh
```

## 部署

```bash
cp keepalived-haproxy/config.example.env keepalived-haproxy/prod.env
vim keepalived-haproxy/prod.env

bash keepalived-haproxy/scripts/deploy-keepalived-haproxy.sh keepalived-haproxy/prod.env
```
