# K3s HA 文档索引

本目录覆盖 3 台 master 节点的 K3s 高可用生产部署。

部署方案覆盖：

- `amd64` / `arm64` 主机。
- cgroup v1 / cgroup v2 主机预检。
- 在线安装与离线 airgap 安装。

K3s 部署本身不依赖 VIP/HAProxy。三台 master 可以先通过 master1 的 `6443` 完成集群引导；如果后续需要统一 API 入口，再单独部署 [../keepalived-haproxy/说明.md](../keepalived-haproxy/说明.md) 或外部 LB。

阅读顺序：

1. [docs/01-架构说明.md](docs/01-架构说明.md)：拓扑、端口、故障模型、生产前置条件。
2. [docs/02-在线部署.md](docs/02-在线部署.md)：在线一键部署。
3. [docs/03-离线部署.md](docs/03-离线部署.md)：离线包准备与离线部署。
4. [docs/04-运维手册.md](docs/04-运维手册.md)：巡检、备份、故障处理、升级原则。

脚本入口：

```bash
bash k3s/scripts/deploy-k3s-ha.sh k3s/prod.env
```
