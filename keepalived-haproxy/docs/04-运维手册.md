# 运维检查

## 服务状态

```bash
systemctl status haproxy keepalived
ss -lntp | grep 8443
```

## VIP

```bash
ip addr show <NODE_INTERFACE> | grep <VIP>
journalctl -u keepalived -n 100 --no-pager
```

任意时刻 VIP 只应出现在一台节点上。

## HAProxy

```bash
haproxy -c -f /etc/haproxy/haproxy.cfg
journalctl -u haproxy -n 100 --no-pager
```

如果 `VIP:8443` 不通，按顺序检查：

1. VIP 是否漂移到某台节点。
2. HAProxy 是否监听 `VIP:8443`。
3. 后端 `backend:6443` 是否可达。
4. 防火墙是否阻断 VRRP 或 TCP 端口。
