# Rook Ceph

本目录描述在 3 master K3s 集群上部署 Rook Ceph，并启用：

```text
CephFS  -> Kubernetes ReadWriteMany 共享文件系统
RGW     -> S3 兼容对象存储
```

推荐最小生产形态：

```text
3 台 K3s master
每台 1 块独立裸数据盘
Rook Operator
Ceph MON x 3
Ceph MGR x 2
Ceph OSD x 3
Ceph MDS active + standby
RGW x 2
副本数 size=3
故障域 host
```

不要用系统盘目录模拟 OSD 作为生产方案。Rook Ceph 是基础设施级资源，后续 Gitea、Harbor 等目录只引用本目录创建出来的 `StorageClass` 或 S3 endpoint。

## 部署

```bash
cp rook-ceph/config.example.env rook-ceph/prod.env
vim rook-ceph/prod.env

bash rook-ceph/scripts/deploy-rook-ceph.sh rook-ceph/prod.env
```

离线环境先准备并上传镜像：

```bash
bash rook-ceph/scripts/download-rook-ceph-assets.sh rook-ceph/prod.env
bash rook-ceph/scripts/upload-rook-ceph-images.sh rook-ceph/prod.env
bash rook-ceph/scripts/deploy-rook-ceph.sh rook-ceph/prod.env
```
