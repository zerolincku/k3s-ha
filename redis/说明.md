# Redis Sentinel

本目录描述在 3 master K3s 集群上部署 Redis Sentinel 高可用。

Redis Sentinel 不是 3 个可写 master。它的拓扑是：

```text
1 个 Redis master
2 个 Redis replica
3 个 Redis Sentinel
```

Sentinel 负责监控 master、触发故障转移、让客户端发现当前 master。本方案按用户要求不启用持久化，适合缓存、临时会话、可重建数据等场景；不适合保存不可丢失的数据。

```bash

vim redis/prod.env

bash redis/scripts/deploy-redis-sentinel.sh redis/prod.env
```
